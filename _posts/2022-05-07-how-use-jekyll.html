安装

1 推荐使用这个模板：https://github.com/cotes2020/jekyll-theme-chirpy/

2. 创建项目

也可以使用下面的步骤（生成的页面是空白图）

1. gem install jekyll

2. jekyll new <username>.github.io

3. bundle add webrick

4. jekyll serve --host 0.0.0.0

打开浏览器， localhost:4000 即可

配置

修改文件 _config.yml 即可，把里面的用户名啥的都改掉。

使用

把文章内容放到 _post 目录下， 格式为： 2022-05-07-welcome-to-jekyll.markdown

内容要按照md 格式来。

也可以插入html

顶部的layout 非常重要。 category可以不写。 写了之后会在URL的地方生成
