go 默认是从github.com上引用的.不过由于种种原因,这个不符合我国国情

而且我们开发的时候都要从本地引入

方法五花八门,版本也各有区别

所以,我们简单一些:

1. 创建一个文件夹,  $ mkdir test_go_mod

$ cd test_go_mod

$ go mod init test_go_mod (这里指明 该package 跟所在目录文件名一样)

$ go mod tidy

然后,我们创建一个main.go文件:

package main

import (
  "fmt"
  "test_go_mod/kaka"
)

func main(){

  fmt.Println("lalala")
  kaka.Say()
}

同时, 创建 kaka/kaka.go 

$ cat kaka/kaka.go
package kaka

func Say() string{
  return "hihihi"
}

最后.我们看一下 test_go_mod/go.mod 文件 (不用修改它) 

module test_go_mod

go 1.17

最后,回顾一下当前文件夹路径( 一定要留意main.go所在的文件夹名称)

test_go_mod/
test_go_mod/kaka
test_go_mod/kaka/kaka.go
test_go_mod/go.mod
test_go_mod/main.go

最后,就可以使用了.

/workspace/test_go_mod$ go run main.go
lalala
