今天安装的时候报错了。找了下日志，发现 是这个原因：

error: macro "EVP_MD_CTX_create" passed 1 arguments, but takes just 0
EVP_MD_CTX *EVP_MD_CTX_create(void);

解决办法
1。 sudo apt purge libssl-dev && sudo apt install libssl1.0-dev

2.  $ rbenv install 2.3.8 

就可以了。
