# Yates 的个人博客

https://iamyates.com

使用了Jekyll （部分使用）

## 使用

```
$ ruby -v # 3.x
$ bundle install
$ bundle exec jekyll serve --host 0.0.0.0
```

会生成一个 `_site` 文件夹，上传到远程即可。


## 使用
